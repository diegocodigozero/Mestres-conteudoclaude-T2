Squads de Agentes

Times de IA que trabalham por você. Cada squad é um pequeno exército de agentes especializados — você cola um prompt, configura o que for preciso e digita iniciar squad. O resto roda sozinho.

10 squads 56 agentes 52 skills
você@claude~/squads $iniciar squad
Conceito

O que é um squad?

Pense num time de trabalho. Em vez de você fazer tudo, um orquestrador distribui as tarefas para agentes especialistas, e cada agente segue suas skills (o passo a passo de como fazer). Você aprova o resultado no final.

Orquestrador

O maestro do time. Faz o onboarding com você, decide a ordem e aciona os agentes certos. Ele coordena — não faz o trabalho braçal.

Agentes

Especialistas com uma função cada (coletar, analisar, redigir...). Trabalham isolados e entregam sua parte. Vivem em .claude/agents/.

Skills

Os manuais de "como fazer" que os agentes seguem — o passo a passo técnico de cada tarefa. Vivem em .claude/skills/.

você→ orquestrador→ agentes+ skills→ resultado pronto para você aprovar
Anatomia

A hierarquia de pastas de um squad

Todo squad segue a mesma estrutura. Entender essas pastas é o suficiente para saber onde tudo mora — e onde você coloca suas chaves e dados.

nome-do-squad/
├─CLAUDE.mdo manual-mãe: o que o squad é e as regras
├─prompt-montagem.mdo prompt que monta o squad — você cola isso no Claude Code
├─config/
│  ├─.env.examplemodelo das chaves (versionado, vazio)
│  └─.envsuas chaves de verdade — nunca versionado
├─.claude/
│  ├─agents/os especialistas — 1 arquivo .md por agente
│  └─skills/os "como fazer" — 1 pasta por skill (com SKILL.md)
└─output/tudo que o squad gera fica aqui
CLAUDE.md

O cérebro do projeto. O Claude Code lê esse arquivo automaticamente e entende o que o squad faz e como se comportar.

prompt-montagem.md

O gerador do squad. Você copia o conteúdo dele e cola no Claude Code — ele cria os agentes, as skills e o resto sozinho.

config/.env

Onde ficam suas chaves de API (quando o squad precisa). Nunca é versionado — fica só na sua máquina.

.claude/agents · skills

O coração do squad: os agentes (especialistas) e as skills (os manuais técnicos que eles seguem).
