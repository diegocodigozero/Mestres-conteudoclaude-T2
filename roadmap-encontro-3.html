Configurações Críticas do GitHub

Depois de criar a conta, faz isso primeiro.

GitHub sem essas 3 configurações = Vercel e Supabase recusando teu login. 5 minutos pra evitar trava no encontro.

01
OBRIGATÓRIO

Confirmar email

2 MIN
  • Vai no email que você cadastrou.
  • Clica no link de confirmação que o GitHub mandou.
  • Sem isso, Vercel e Supabase não aceitam login com tua conta.
02
OBRIGATÓRIO

Ativar 2FA (autenticação em 2 fatores)

3 MIN
  • Vai em Settings → Password and authentication → Two-factor authentication.
  • Escolhe "Authenticator app" (Google Authenticator, 1Password ou Authy).
  • Sem 2FA, o GitHub pode bloquear pushes e integrações em algum momento.
  • Guarda os códigos de recuperação num lugar seguro (1Password ou anotação física).
03
RECOMENDADO

Configurar email privado

2 MIN
  • Vai em Settings → Emails.
  • Marca "Keep my email addresses private".
  • Marca "Block command line pushes that expose my email".
  • Isso protege teu email real quando o Claude Code fizer commits.
